These are some projects I have worked on

Feed Me Recipes
Feed Me Recipes is a Facebook Messenger Chatbot that allows users to search for recipes with a list of ingredients they have or want to use.
A user will also be able to rate different recipes and the system will learn to return recipes with similar ingredients.
I wanted to expand my knowledge into Machine Learning Topics such as Natural Language Processing, as well as Neural Networks.

Developed using: Python, TensorFlow, React.js, Spoonacular API

Project Runaway
Project Runaway is an iOS application with the goal of keeping drivers on the road aware of the weather along their roadtrip route
I wanted to learn about iOS application development and refine my skills in Node.js development

Developed using: swift, Node.js, Google Maps API, Dark Sky API

Crâvo
Crâvo is a shared economy Android application strictly targetted at having shoppers at Wal-Mart pick up items for users who are located on the shopper's way home.
I wanted to learn about Android application development as well as learn about how Node.js works

Developed using Java, Node.js, Google Maps API, Walmart Search API

Climate Change Tracker
Climate Change Tracker allows users to see the temperature of any city, state or country over the past 300 years. It also allows users to see 100 years into the future and how the temperature might be.
I wanted to learn about Angular.js as well as practice using multiple linear regressions that I learned in my statistics class.

Developed using: Angular.js, Node.js

CySLI Rocket Simulator
CySLI Rockect Simulator is a simulation space we used to test algorithms for when to engage the air brakes on the rocket to reach a final apogee of one mile.
We would connect our flight computer (arduino duo), loaded with the algorithms to our simulator, to run real time simulations.
I wanted to expand my knowledge of embedded systems from the classroom into an actual project as well as get more experince in C++.

Developed using: C++, Embedded C
